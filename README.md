# Smart Task Manager 🚀

A simple task management web application built using Flask.

## 🔥 Features
- User Registration & Login
- Add, Delete, Complete Tasks
- Priority & Deadline Tracking
- Clean UI

## 🛠 Tech Stack
- Python (Flask)
- SQLite
- HTML, CSS

## ▶️ How to Run
1. Install Python
2. Run:
   python app.py
3. Open browser:
   http://127.0.0.1:5000
